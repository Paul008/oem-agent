<objective>
## Goal
Create a `/dashboard/style-guide/:oemId` page that renders a read-only visual brand catalog for any OEM — showing their brand tokens (colors, typography, spacing) and all recipes rendered as live previews.

## Purpose
Stakeholders need a visual reference for each OEM's design system. Currently brand tokens and recipes are spread across database tables with no unified visual view. This page is the "brand book" that shows how an OEM's pages should look.

## Output
- New dashboard page at `/dashboard/style-guide/:oemId`
- New worker endpoint `GET /admin/style-guide/:oemId` returning brand tokens + recipes in one payload
- Sidebar nav entry under Infrastructure
- OEM selector to switch between brands
</objective>
