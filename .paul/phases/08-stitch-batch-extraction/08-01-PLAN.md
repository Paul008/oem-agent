<objective>
## Goal
Add batch recipe extraction — extract recipes from multiple OEM URLs at once instead of one at a time. Stitch MCP integration deferred to a follow-up (requires external setup).

## Purpose
Currently users extract one URL at a time. Batch extraction lets them queue multiple pages (e.g., homepage + model pages) and get all recipes in one operation.

## Output
- Batch extraction UI: add multiple URLs, extract all sequentially
- Results grouped by URL
- Stitch MCP deferred (needs external configuration)
</objective>
